Daily Roulette

Comment ça marche ?

1

Ajoutez votre équipe

Saisissez les prénoms de chaque participant dans l'écran de configuration, puis validez avec Ajouter.

2

Lancez le Daily

Cliquez sur Lancer le Daily pour passer à la vue roulette. L'équipe est sauvegardée automatiquement dans votre navigateur.

3

Tirez au sort

Appuyez sur Lancer pour une animation de type machine à sous qui désigne aléatoirement le prochain participant. Vous pouvez aussi utiliser Passer pour tirer sans animation.

4

Suivez la progression

La barre de progression et la grille des participants vous montrent en temps réel qui a déjà parlé. Une fois tout le monde passé, cliquez sur Replay pour relancer un tour.

5

Gérez votre équipe

Utilisez le bouton Gérer l'équipe pour ajouter ou retirer des participants à tout moment.
